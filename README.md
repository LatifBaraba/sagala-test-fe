# Sagala Front End Test

by Abdullatif Ali

## Build with

- React with Vite Js
- Node v18.16.0
- PNPM

## Libraries

### UI & Components

- Tailwind Css
- Chakra UI

### State Management

- Zustand

### Validation

- React Hook Form
- ZOD

### Table

- React Table (by Tanstack)

### Icons

- React Icons
- Lucide React

### Date

- Dayjs

## Installation

Use the package manager npm/yarn/pnpm.

```bash
pnpm i
```

## Run Application

Use the package manager npm/yarn/pnpm.

```bash
pnpm dev
```

Or open this link [Sagala Test Latif](https://pip.pypa.io/en/stable/)
